# MPRIORITY-2.0

Веб-версия программы для поддержки принятия решений на основе метода анализа иерархий (МАИ/AHP) Томаса Саати.

## Структура проекта

- `backend/` - Express API сервер для расчетов AHP
- `frontend/` - Next.js веб-приложение

## Развертывание

- **Backend**: Railway
- **Frontend**: Vercel

## Функциональность

- Построение иерархии (Цель → Критерии → Альтернативы)
- Парные сравнения критериев и альтернатив
- Расчет приоритетов и весовых коэффициентов
- Проверка согласованности суждений
- Визуализация результатов
