# MPRIORITY 2.0 Frontend

Next.js веб-приложение для метода анализа иерархий (AHP).

## Функциональность

- Построение иерархии (Цель → Критерии → Альтернативы)
- Парные сравнения критериев и альтернатив
- Проверка согласованности суждений
- Визуализация результатов (графики, таблицы)
- Экспорт результатов в JSON

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение запустится на http://localhost:3000

## Переменные окружения

Создайте файл `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:3001
```

Для продакшена укажите URL вашего Railway backend.
