# Быстрый старт MPRIORITY 2.0

## Локальный запуск (для тестирования)

### 1. Запуск Backend

```bash
cd backend
npm install
npm start
```

Backend будет доступен на http://localhost:3001

### 2. Запуск Frontend

В новом терминале:

```bash
cd frontend
npm install
npm run dev
```

Frontend будет доступен на http://localhost:3000

### 3. Использование

1. Откройте http://localhost:3000 в браузере
2. Введите цель (например: "Выбор ноутбука")
3. Добавьте критерии (минимум 2, например: "Цена", "Производительность", "Вес")
4. Добавьте альтернативы (минимум 2, например: "Модель A", "Модель B", "Модель C")
5. Сравните критерии попарно
6. Сравните альтернативы по каждому критерию
7. Просмотрите результаты с графиками и таблицами
